Creating an Instance

// Path to the RavenAPI class
require_once 'path/to/raventools-api-php/src/class.raven-api-php.php';
// Instance to use for calls
$Raven = new RavenToolsAPI('YOUR_API_KEY');

Rank (method: rank)

This request will return a list of matches for a particular domain, keyword, search engine, and date range. You can only access results for domains and keywords that have been added to your account, including competitor domains.

// Helper Method
$rank = $Raven->getRank( $domain, $keyword, $start_date, $end_date, $engine );
// Direct Call
$rank = $Raven->get('rank', array(
	'domain'=>$domain,
	'keyword'=>$keyword,
	'start_date'=>$start_date,
	'end_date'=>$end_date,
	'engine'=>'all'
	));

Rank All (method: rank_all)

This request will return a list of matches for a particular domain and date. You can only access results for domains and keywords that have been added to your account, including competitor domains.

// Helper Method
$rank_all = $Raven->getRankAll( $domain, $start_date, $end_date );
// Direct Call
$rank_all = $Raven->get('rank_all', array(
	'domain'=>$domain,
	'start_date'=>$start_date,
	'end_date'=>$end_date
	));

Domains (method: domains)

This request will return the available domains for the profile associated with your API key.

// Helper Method
$domains = $Raven->geDomains();
// Direct Call
$domains = $Raven->get('domains');

Rank Max Week (method: rank_max_week)

This request returns the ISO Week number (YYYYWW) and date (YYYY-MM-DD) for the latest week with complete results for all keywords in a domain or a domain/keyword pair. It returns null for date/week and status = 'no data' for domains or domain/keywords that have no available data.

// Helper Method
$rank_max_week = $Raven->getRankMaxWeek( $domain, $keyword );
// Direct Call
$rank_max_week = $Raven->get('rank_max_week', array(
	'domain'=>$domain,
	'keyword'=>$keyword
	));

Engines (method: engines)

This request will return the available search engines for tracking keywords, to be used when adding or modifying domains.

// Helper Method
$engines = $Raven->getEngines();
// Direct Call
$engines = $Raven->get('engines');

Profile Info (method: profile_info)

This request will return the name and billable keyword usage for the current profile.

// Helper Method
$profile_info = $Raven->getProfileInfo();
// Direct Call
$profile_info = $Raven->get('profile_info');

Domain Info (method: domain_info)

This request will return the search engines for the domain provided.

// Helper Method
$domain_info = $Raven->getDomainInfo();
// Direct Call
$domain_info = $Raven->get('domain_info');

Competitors (method: competitors)

// Helper Method
$competitors = $Raven->getCompetitors( $domain )
// Direct Call
$competitors = $Raven->get('competitors', array('domain'=>$domain));

Keywords (method: keywords)

This request will return the available keywords for the domain provided.

// Helper Method
$keywords = $Raven->getKeywords( $domain );
// Direct Call
$keywords = $Raven->get( 'keywords', array('domain'=>$domain) );

Keywords (method: get_links)

This request will return the all links for the domain provided.

// Helper Method
$keywords = $Raven->getLinks( $domain )
// Direct Hall
$links = $Raven->get( 'get_links', array('domain'=>$domain) );
